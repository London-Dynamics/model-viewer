LD Floating Object Anchor by London Dynamics. The floating-object-anchor slot tracks the selected object's bounding sphere in screen space. The slot is centered on the sphere; data-sphere-radius-px reports the projected radius in pixels.

Bounding sphere visualization

slot="floating-object-anchor"

Place a modular object on the floor, select it, and orbit the camera. The dashed circle follows the projected bounding sphere. A future step is to pair this anchor with Floating UI, using the center as the reference point and the radius as the placement offset.

Floating UI with radius offset

placement: "bottom" with offset: sphereRadiusPx

This uses an arbitrary HTML panel with Floating UI. The slot center is the reference, and data-sphere-radius-px is used as offset.
