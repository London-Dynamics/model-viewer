LD History records committed transforms, placements, deletions, and group/snap operations. Subscribe to history-change to drive toolbar UI.

Undo / Redo

API buttons nudge the purple Box mesh by name (no selection needed). Drag in the scene still records undo steps when using placed parts in modular scenes. Use the toolbar buttons or call undo() / redo() from your host app.
