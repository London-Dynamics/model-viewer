Viewport gizmo positioning

Control the viewport gizmo position from outside <model-viewer> using CSS custom properties on the element. You can use em, rem, or any CSS length for offsets.

Interaction mode: rotate vs pan

Toggle interaction-mode between rotate and pan, then switch camera type to verify pan mode keeps interactions enabled even when camera-controls is not set.

Camera JSON interactions

Save and restore camera state. Persist getCameraJSON().object in your host (include controlsSnapshot for views after pan/truck). Restore with await setCameraFromJSON(saved). Legacy controlsState strings still work. Use Get Camera JSON to inspect the payload; round-trip buttons run silent checks.
