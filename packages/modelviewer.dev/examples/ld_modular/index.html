LD Modular by London Dynamics. This page showcases various utility functions used by Client-side Puzzler.

Place a GLB into the scene

placeGLB() – adds more geometry to the scene.

This example uses CSS parts to style the default UI elements from the light DOM. Check the CSS in the page head to see how ::part() selectors work.

Transforms

Call rotate(), translate(), and scale() with an object's name to manipulate its transform. The name "scene" can be used to manipulate the entire model.

Rotation Mode

Call rotate(), with an object's name to manipulate its rotation. The name "scene" can be used to manipulate the entire model. The rotation mode can be set to "relative" or "snapToClosest".

Buffer, not SRC!

setSrcFromBuffer() – Update the source from an ArrayBuffer instead of URL.

Bulk Placement

placeManyGlb() – place multiple GLBs at once using known transforms.
