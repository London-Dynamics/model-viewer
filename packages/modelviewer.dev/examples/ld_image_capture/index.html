LD Image Capture by London Dynamics. Capture the current view as a data URL with optional width, height, file type, camera pose, and crop using captureImage().

Capture Image

Capture the current view as PNG or JPEG. The data URL is shown in the console and can be used as an image src.

Capture with camera pose

Pass a camera state (e.g. from getCameraJSON()) into captureImage({ camera }). The view is updated to that pose, one frame is rendered, then the image is captured.

Capture with crop

Use crop: { x, y, width, height } (in source canvas pixels) to capture only a region. Optionally combine with width and height to scale the crop.

Background color

Use backgroundColor so formats like JPEG (no transparency) show a solid background instead of black. Default is white; here it is set to cornflowerblue.
